# UX экраны и потоки

## Клиентское приложение (iOS/Android)
Экраны:
1) Онбординг / выбор города (в MVP сразу Istanbul)
2) Главная: услуги + популярные мастера + кнопка «Записаться»
3) Услуги: список карточек с длительностью и ценой
4) Мастера: карточки мастеров + рейтинг + фильтр «первый свободный»
5) Календарь: выбор даты
6) Время: список слотов
7) Подтверждение записи
8) Мои записи (активные + история)
9) Профиль (телефон, имя)

Основной поток:
Главная → Услуга → Мастер → Дата → Время → Подтверждение → Мои записи

Доп. сценарии:
- Быстрая запись: Услуга → Первый свободный мастер → Время
- Отмена/перенос: Мои записи → Детали → Отменить/Перенести

## Веб‑панель мастера
Экраны:
1) Вход (телефон/код)
2) Календарь дня (тайм‑лайн + список)
3) Запись (детали клиента, услуга, время)
4) Управление временем (перерывы/выходные)
5) Календарь недели/месяца (перенос записей)

Основной поток:
Календарь → Запись → Подтвердить/Отменить/Предложить другое время

## Веб‑панель администратора
Экраны:
1) Дашборд (сегодняшние записи + загрузка мастеров)
2) Мастера (добавить/редактировать)
3) Услуги и цены
4) График работы салона
5) Все записи (поиск и фильтры)
6) Календарь недели/месяца (перенос записей)

Роли:
- Админ видит всех мастеров и записи
- Мастер видит только свои записи
- Клиент видит только свои записи
