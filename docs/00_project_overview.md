# Istanbul — онлайн-запись

Цель: сделать понятный демо‑продукт онлайн‑записи для парикмахерской Istanbul (Казань), чтобы показать владельцу и запустить пилот.

Состав демо:
- Мобильное приложение для клиентов (iOS/Android)
- Веб‑панель для мастеров
- Веб‑панель для администратора

Основные сценарии:
1) Клиент записывается: услуга → мастер → время → подтверждение
2) Клиент отменяет/переносит запись
3) Мастер видит календарь, подтверждает/отменяет запись
4) Админ управляет услугами, мастерами и графиком работы

Ключевые ограничения MVP:
- Оплата не требуется
- Один филиал
- 6 мастеров
- График работы до 22:00

Документы в папке /docs:
- Схема БД и демо‑данные
- API‑контракты
- UX‑экраны и потоки
- Алгоритм свободных слотов
- Тексты для лендинга
- Питч на 6 слайдов
- Роли и политики доступа
